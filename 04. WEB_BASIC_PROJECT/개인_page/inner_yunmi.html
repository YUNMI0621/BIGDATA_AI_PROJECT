✍ 과정 공부 계기 ✍

 빅데이터 분석가 양성과정에 참여하게 된 계기

 통계학과 1학년때부터 빅데이터와 관련된 중요성은 수 없이 많이 들었습니다.
 또한 금융이나 보험쪽을 좋아해서 2학년때 금융보험 복수전공을 신청하게
되었습니다.

 금융이나 보험 관련 취업의 길이 점점 좁아지고 있어 폭을 넓히기 위해
 빅데이터와 접목시키면 좋을 것 같아 ADSP, SQLD,
 사회조사분석사등 데이터 관련 자격증을 땄습니다.

 물론 자격증과 같은 이론적인 부분도 중요하지만 실무에서 활용할 수 있는
 경험이나 직접 데이터를 이용하여 접해보는 것이 더 중요하다는 것을
 느끼게 되어 빅데이터분석가 양성과정에 참여하게 되었습니다.

📗 분야에 대한 간단한 소개 📗



 관심있는 분야1

 ▪ 데이터 전처리와 시각화를 활용한 분석
 - 빅데이터를 활용한 데이터 분석 및 시각화를 통한 결론 분석이
  흥미로웠고 처음 생각과는 다른 결과가 나온다는 점도 재밌었습니다.

 관심있는 분야2

 ▪ 머신러닝, 딥러닝을 활용한 모델 생성
 - 사람이 먼저 컴퓨터에 특정 패턴을 추출하는 방법을 지시하고, 그 이후
 컴퓨터가 스스로 데이터의 특징을 분석하고 축적된 데이터를 바탕으로 문제를 해결한다는 점이 새로운 재미로 다가왔습니다.
 - 머신러닝과는 조금 차이가 있지만 다르게 딥러닝은 컴퓨터가 스스로
 데이터를 기반으로 학습할 수 있도록 정해진 신경망을 컴퓨터에게 주고
 경험을 중심으로 학습을 수행한다는 점이 좋았습니다.

 관심있는 취업 분야1

 ▪ 금융보험관련
 이유 : 대학교2학년 때부터 공부를 시작하게 되어 흥미가 생김에 따라
